About the Extension

The Parallel Reading extension helps users learn languages by providing side-by-side translations of web content. It's designed to work with both English and Spanish content.

To test the extension:

  1. Install the extension in your browser
  2. Navigate to this test page
  3. Click the extension icon
  4. Click the "Translate" button
  5. The sidebar should open with parallel translations
